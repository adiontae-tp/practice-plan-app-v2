TAB SELECTOR
================================================================================

+----------------------------------------------------+
|  [Period Usage]  [Tag Usage]                       |
+----------------------------------------------------+

TPToggleGroup with 2 columns:
- Period Usage: Shows period-based analytics
- Tag Usage: Shows tag-based analytics

Selected tab has blue background
Unselected tab has gray outline


PERIOD USAGE TAB
================================================================================

Summary Cards:
+----------+  +----------+  +----------+
|    12    |  |   18h    |  |  1.5h    |
| Practices|  | Total    |  | Average  |
|          |  | Time     |  | Duration |
+----------+  +----------+  +----------+

Period Breakdown Table:
+-------------------+------+------------+--------+
|   Period Name     | Uses | Total Time | Avg    |
+-------------------+------+------------+--------+
| Warm-up           |  12  |   3h 0m    | 15m    |
+-------------------+------+------------+--------+
| Skill Drills      |  10  |   5h 0m    | 30m    |
+-------------------+------+------------+--------+

Columns:
- Period Name: Name of period template
- Uses: Number of times used
- Total Time: Sum of all usages
- Avg: Average duration per use

INTERACTIONS
================================================================================

1. Click tab -> Switch between Period/Tag usage
2. Click table row -> Could show drill-down details
3. Click [Export] -> Export report as PDF/CSV
4. Change date range -> Filter data
