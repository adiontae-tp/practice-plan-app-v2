ANNOUNCEMENT CARD

+--------------------------------------------------+

+--------------------------------------------------+
|     Announcement Title                     [MED] |  No dot, normal weight

+--------------------------------------------------+


UNREAD INDICATOR
================================================================================

[*] = Blue dot (8px)
- Position: Left of title
- Color: #356793 (primary blue)
- Only shown for unread announcements


PRIORITY BADGES
================================================================================

[HIGH] = Red background (#ef4444), white text
[MED]  = Pink background (#ef7b8f), white text
[LOW]  = Gray background (#9ca3af), white text

Badge styling:
- Border radius: 4px
- Padding: 4px 12px
- Font size: 11px
- Font weight: 600
- Uppercase text


TIMESTAMP FORMAT
================================================================================

- Just now: "just now"
- < 1 hour: "X minutes ago"
- < 24 hours: "X hours ago"
- < 7 days: "X days ago"
- < 4 weeks: "X weeks ago"
- > 4 weeks: "MMM D, YYYY"

CARD STATES
================================================================================

Default:
- White background
- Normal shadow

Hover:
- Slightly elevated shadow
- Cursor pointer

Unread:
- Title: Bold font weight
- Blue dot visible

Read:
- Title: Normal font weight
- No dot


INTERACTIONS
================================================================================

1. Click card -> Open announcement-detail modal
   - Marks as read if unread
2. Click [+ New Announcement] -> Open announcement-create modal
3. Hover card -> Elevate card slightly


PERMISSIONS
================================================================================

Admin users:
- See [+ New Announcement] button
- Can create/edit/delete announcements

Non-admin users:
- No button visible
- View-only access
- Can mark as read


CONTENT AREA WIDTH
================================================================================

- Max width: max-w-5xl (1024px)
- Centered in content area
- Responsive padding
