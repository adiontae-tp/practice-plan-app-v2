LOGOUT BUTTON
================================================================================

+--------------------------------+
|          [Logout]              |
+--------------------------------+

- Full width in card
- Red background (#ef4444)
- White text
- Shows confirmation before logout


LOGOUT CONFIRMATION
================================================================================

+------------------------------------------+
|  +------------------------------------+  |
|  |           Log Out?                 |  |
|  |                                    |  |
|  |  Are you sure you want to          |  |
|  |  log out of your account?          |  |
|  |                                    |  |
|  |  [Cancel]           [Log Out]      |  |
|  +------------------------------------+  |
+------------------------------------------+


EDIT MODE FOOTER
================================================================================

+--------------------------------+
|  [Cancel]             [Save]   |
+--------------------------------+

[Cancel] = Discard changes, return to view mode
[Save] = Save changes, return to view mode


INTERACTIONS
================================================================================

View Mode:
1. Click [Edit Profile] -> Enter edit mode
2. Click [Copy] -> Copy ID to clipboard
3. Click [Logout] -> Show confirmation

Edit Mode:
1. Edit first/last name fields
2. Click photo buttons -> Camera/gallery/remove
3. Click [Save] -> Save and exit edit mode
4. Click [Cancel] -> Discard and exit edit mode


VALIDATION
================================================================================

First Name: Required, min 1 character
Last Name: Required, min 1 character
