TABLE STRUCTURE
================================================================================

+------+------------------+----------+-----------+--------+--------+
|  #   |      Name        | Duration |   Tags    | Notes  | Actions|
+------+------------------+----------+-----------+--------+--------+
|  1   | Warm-up          |  15 min  | Cond.     |  [N]   |  [X]   |
|      |                  |          | Stretch   |        |        |
+------+------------------+----------+-----------+--------+--------+

Columns:
- # : Row number (width: 40px)
- Name: Period name (flex-1)
- Duration: Formatted as "Xh Ym" or "X min" (width: 120px, center)
- Tags: Comma-separated list (flex-1)
- Notes: Icon indicator [N] or dash (width: 100px, center)
- Actions: Delete button (width: 80px)


COLUMN DETAILS
================================================================================

Duration format:
- Under 60 min: "30 min"
- 60+ min: "1h 30m"

Tags display:
- Comma-separated: "Conditioning, Stretch"
- Truncate if too long with "..."

Notes indicator:
- [N] = FileText icon (has notes)
- "-" = No notes

ROW CLICK ACTION
================================================================================

Clicking a row opens the Period modal for viewing/editing


DELETE ACTION
================================================================================

[X] = Trash icon button

SEARCH FUNCTIONALITY
================================================================================

+------------------------------------------+
| [O] Search by name, tag, or notes...     |
+------------------------------------------+

- Real-time filtering
- Searches: name, tags, notes content
- Filters both sections simultaneously
- Shows "No results" if no matches


NEW PERIOD BUTTON
================================================================================

[+ New Period]

- Primary blue button
- Opens Period modal in create mode
- Only visible if canEdit


INTERACTIONS
================================================================================

1. Type in search -> Filter table rows
2. Click row -> Open period modal for view/edit
3. Click [X] delete -> Show confirmation, then delete
4. Click [+ New Period] -> Open period modal for create
5. Hover row -> Highlight row


PERMISSIONS
================================================================================

canEdit = true:
- Show [+ New Period] button
- Show delete [X] buttons on user periods
- Allow editing periods

canEdit = false:
- Hide [+ New Period] button
- Hide delete buttons
- View-only mode
