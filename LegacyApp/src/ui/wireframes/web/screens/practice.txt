INTERACTIONS
================================================================================

1. Click "Edit Practice" -> Opens plan/[id] modal
2. Click "View PDF" -> Downloads/opens PDF
3. Click "Share" -> Opens share dialog
4. Click "Delete" -> Opens delete confirmation
5. Click "+ CREATE PRACTICE" -> Opens plan/new modal
6. Click activity row -> Opens activity detail/edit
7. Click [E] edit button -> Opens activity-instance-edit modal


RESPONSIVE NOTES
================================================================================

Desktop (>=768px):
- Table view for activities
- Side-by-side panels
- Quick actions visible

Tablet (768px-1024px):
- May stack panels vertically
- Maintain table view

The component switches to PracticeMobile below 768px breakpoint.
