TABLE HEADER
================================================================================

+----------------------------------------------------+
| [Search by email...]              [+ Invite Coach] |
+----------------------------------------------------+

- Search input for filtering by email
- Primary action button for inviting coaches


TABLE STRUCTURE
================================================================================

+------+---------------------------+------------+--------+
|  #   |          Email            | Permission | Actions|
+------+---------------------------+------------+--------+
|  1   | john@example.com          |   [Admin]  |  [X]   |
+------+---------------------------+------------+--------+

Columns:
- # : Row number (width: 40px)
- Email: Coach email address (flex-1)
- Permission: Permission badge (width: 120px, center)
- Actions: Delete button (width: 80px)


PERMISSION BADGES
================================================================================

[Admin] = Blue background (#356793), white text
          Full access - can manage everything

[Edit]  = Green background (#22c55e), white text
          Can create/edit content

[View]  = Gray background (#9ca3af), white text
          Read-only access

Badge styling:
- Border radius: 4px
- Padding: 4px 12px
- Font size: 11px
- Font weight: 600
- Centered in column

SEARCH FUNCTIONALITY
================================================================================

+------------------------------------------+
| [O] Search by email...                   |
+------------------------------------------+

- Real-time filtering by email address
- Can also search by permission level


INTERACTIONS
================================================================================

1. Type in search -> Filter table rows
2. Click row -> Open edit coach modal
3. Click [X] delete -> Show confirmation, then remove
4. Click [+ Invite Coach] -> Open invite modal
5. Hover row -> Highlight row


PERMISSIONS
================================================================================

Admin users only:
- See [+ Invite Coach] button
- See delete [X] buttons
- Can edit permissions
- Can remove coaches

Non-admin:
- This screen not accessible

Body rows:
