TABLE STRUCTURE
================================================================================

+------+--------------------+----------+------------+--------+--------+
|  #   |       Name         | Duration | Activities |  Tags  | Actions|
+------+--------------------+----------+------------+--------+--------+
|  1   | Monday Practice    |  1h 30m  |     6      | Cond.  |  [X]   |
|      |                    |          |            | Skills |        |
+------+--------------------+----------+------------+--------+--------+

Columns:
- # : Row number (width: 40px)
- Name: Template name (flex-1)
- Duration: Total time formatted (width: 120px, center)
- Activities: Count number (width: 120px, center)
- Tags: Comma-separated (flex-1)
- Actions: Delete button (width: 80px)


COLUMN DETAILS
================================================================================

Duration format:
- Under 60 min: "45 min"
- 60+ min: "1h 30m" or "2h"

Activities:
- Simple count number
- Centered in column

Tags display:
- Comma-separated list
- Truncate with "..." if too long

SEARCH FUNCTIONALITY
================================================================================

+------------------------------------------+
| [O] Search by name or tag...             |
+------------------------------------------+

- Real-time filtering
- Searches: template name, tags


INTERACTIONS
================================================================================

1. Type in search -> Filter table rows
2. Click row -> Open template detail modal
3. Click [X] delete -> Show confirmation, then delete
4. Click [+ New Template] -> Open template create modal
5. Hover row -> Highlight row


PERMISSIONS
================================================================================

canEdit = true:
- Show [+ New Template] button
- Show delete [X] buttons
- Allow editing templates

canEdit = false:
- Hide [+ New Template] button
- Hide delete buttons
- View-only mode
