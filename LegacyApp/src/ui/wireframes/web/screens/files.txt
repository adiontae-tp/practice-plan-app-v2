TABLE STRUCTURE
================================================================================

+------------------+----------+------------+--------+--------+--------+
|      Name        | Category | Uploaded By|  Date  |  Size  | Actions|
+------------------+----------+------------+--------+--------+--------+
| Practice         | Document | John S.    | Mar 15 | 2.4 MB |  [X]   |
| Playbook.pdf     |          |            |        |        |        |
+------------------+----------+------------+--------+--------+--------+

Columns:
- Name: File name (flex-1)
- Category: File type category (width: 120px, center)
- Uploaded By: User name abbreviated (width: 150px)
- Date: Upload date formatted (width: 120px)
- Size: File size formatted (width: 100px, center)
- Actions: Delete button (width: 80px)


COLUMN DETAILS
================================================================================

Name:
- Full filename with extension
- Truncate with "..." if too long

Category:
- Document (PDF, DOC, DOCX, TXT)
- Image (JPG, PNG, GIF, etc.)
- Spreadsheet (XLS, XLSX, CSV)
- Video (MP4, MOV, etc.)
- Audio (MP3, WAV, etc.)
- Archive (ZIP, RAR, etc.)
- Other

Uploaded By:
- First name + Last initial
- Example: "John S."

Date:
- Format: "MMM DD" for current year
- Format: "MMM DD, YYYY" for other years

Size:
- < 1 KB: "X B"
- < 1 MB: "X.X KB"
- < 1 GB: "X.X MB"
- >= 1 GB: "X.X GB"

SEARCH FUNCTIONALITY
================================================================================

+------------------------------------------+
| [O] Search by name, description, category|
+------------------------------------------+

- Real-time filtering
- Searches: file name, description, category


INTERACTIONS
================================================================================

1. Type in search -> Filter table rows
2. Click row -> Open/download file or show detail
3. Click [X] delete -> Show confirmation, then delete
4. Click [+ Upload File] -> Open upload modal
5. Hover row -> Highlight row


PERMISSIONS
================================================================================

canEdit = true:
- Show [+ Upload File] button
- Show delete [X] buttons
- Can upload and delete files

canEdit = false:
- Hide [+ Upload File] button
- Hide delete buttons
- Can view and download only
