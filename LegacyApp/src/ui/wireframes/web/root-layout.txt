================================================================================
RESPONSIVE BEHAVIOR
================================================================================

Breakpoint: 768px

< 768px (Mobile):
- Hide sidebar
- Show bottom tab bar
- Stack navigation with headers

>= 768px (Desktop):
- Show top header (full width)
- Show left sidebar
- Content area with Slot
- No tab bar
