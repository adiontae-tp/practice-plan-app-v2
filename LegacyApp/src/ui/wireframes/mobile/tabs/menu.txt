MENU ITEM CARD DETAIL
================================================================================

Standard Item:
+--------------------------------------+
| [O]  Title                           |
|      Description text here           |
+--------------------------------------+

With Badge:
+--------------------------------------+
| [O]  Title                   [Badge] |
|      Description text here           |
+--------------------------------------+

Badge types:
- [!] = Unread indicator (blue dot or number)
- [Admin] = Admin only feature (gray badge)

ADMIN-ONLY ITEMS
================================================================================

Items with [Admin] badge:
- Coaches
- Team Settings

Behavior for non-admin users:
Option 1: Hide the items entirely
Option 2: Show items but grayed out with "Admin Only" badge
