VALIDATION
================================================================================

First Name:
- Required
- Min 1 character

Last Name:
- Required
- Min 1 character

Email:
- Required
- Valid email format
- Not already registered

Password:
- Required
- Min 6 characters

Confirm Password:
- Required
- Must match password

LOADING STATE
================================================================================

+--------------------------------------+
|      [Creating Account...]           |  Spinner
+--------------------------------------+


SUCCESS BEHAVIOR
================================================================================

On successful registration:
1. Firebase account created
2. User document created in Firestore
3. Navigate to onboarding: /(auth)/onboarding/role-select


INTERACTIONS
================================================================================

1. Fill first name -> Updates state
2. Fill last name -> Updates state
3. Fill email -> Updates state
4. Fill password -> Updates state, shows strength
5. Fill confirm password -> Updates state, validates match
6. Tap [O] -> Toggle password visibility
7. Tap [Create Account] -> Validates and submits
8. Tap "Sign In" -> Navigate to login
9. Tap footer links -> Navigate to terms/privacy
