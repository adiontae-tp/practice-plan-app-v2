FOOTER BUTTONS
================================================================================

+------------------------------------------+
|  [Back]                    [Continue]    |
+------------------------------------------+

[Back] = Return to role-select
[Continue] = Go to team-setup (disabled until selection)


INTERACTIONS
================================================================================

1. Tap [<] Back or [Back] -> Return to role-select
2. Type in search -> Filter sports
3. Tap sport cell -> Select that sport
4. Tap [Continue] -> Navigate to team-setup
