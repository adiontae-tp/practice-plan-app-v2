INTERACTIONS
================================================================================

1. Tap [<] Back -> Return to login
2. Enter email -> Updates email state
3. Tap [Send Reset Link] -> Validates and sends
4. Tap "Sign In" link -> Navigate to login

After success:
5. Tap [Resend Email] -> Sends another reset email
6. Tap [Back to Sign In] -> Navigate to login


VALIDATION
================================================================================

Email:
- Required
- Valid email format


FIREBASE BEHAVIOR
================================================================================

Uses Firebase sendPasswordResetEmail:
- Sends email with reset link
- Link expires after set time
- User clicks link -> Opens reset page
- User sets new password
