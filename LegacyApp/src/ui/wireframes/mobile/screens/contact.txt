CONTACT TYPE SELECTOR
================================================================================

+--------------------------------------+
|  [Feedback] [Bug] [Support] [Other]  |
+--------------------------------------+

Toggle button group:
- Single selection required
- Selected: blue background, white text
- Unselected: gray outline, gray text

Types:
- Feedback: General feedback and suggestions
- Bug: Report a bug or issue
- Support: Need help with something
- Other: Anything else


SUBJECT INPUT
================================================================================

+--------------------------------------+
|  SUBJECT                             |
+--------------------------------------+
|  +--------------------------------+  |
|  | Brief description...           |  |
|  +--------------------------------+  |
+--------------------------------------+

- Single line text input
- Placeholder: "Brief description..."
- Required field
- Max 100 characters


MESSAGE INPUT
================================================================================

+--------------------------------------+
|  MESSAGE                             |
+--------------------------------------+
|  +--------------------------------+  |
|  |                                |  |

|  |                                |  |
|  |                                |  |
|  +--------------------------------+  |
|                           (0/1000)   |
+--------------------------------------+

- Multiline textarea (6+ lines)
- Placeholder: "Tell us more..."
- Required field
- Max 1000 characters
- Character count shown


DEVICE INFO TOGGLE
================================================================================

+--------------------------------------+
|  INCLUDE DEVICE INFO                 |
|                                [ON]  |
|                                      |
|  Helps us troubleshoot issues        |
+--------------------------------------+

When ON, includes:
- Device model
- OS version
- App version
- Build number
- User ID (anonymized or with consent)

Toggle default: ON

VALIDATION
================================================================================

Required fields:
- Contact Type (must select one)
- Subject (min 5 characters)
- Message (min 20 characters)

OTHER CONTACT OPTIONS
================================================================================

+--------------------------------------+
|  OTHER WAYS TO REACH US              |
+--------------------------------------+
|                                      |
|  [mail] support@practiceplan.app     |  -> Opens email client
|                                      |
|  [twitter] @PracticePlanApp          |  -> Opens Twitter/X
|                                      |
+--------------------------------------+

Tapping email: Opens default mail app with address
Tapping social: Opens app or browser


INTERACTIONS
================================================================================

1. Tap contact type -> Selects that type
2. Type in subject -> Updates subject field
3. Type in message -> Updates message, shows char count
4. Toggle device info -> Enables/disables inclusion
5. Tap [Send Message] -> Validates and sends
6. Tap email/social -> Opens respective app
7. Tap [<] back -> Returns to Menu (confirms if unsaved)


SENDING STATE
================================================================================

+--------------------------------------+
|       [spinner] Sending...           |
+--------------------------------------+

Button disabled, shows spinner
