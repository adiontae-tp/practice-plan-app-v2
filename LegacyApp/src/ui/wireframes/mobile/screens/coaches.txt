COACH CARD DETAIL
================================================================================

+--------------------------------------+
| [mail]  Coach Display Name           |  Mail icon + derived name
|         coach@email.com              |  Email address
|         [Permission]        [Status] |  Badges
|                                 [X]  |  Delete button
+--------------------------------------+

Display name derived from email:
- "john.smith@email.com" -> "John Smith"
- Uses first part before @ and capitalizes


PERMISSION BADGES
================================================================================

[Admin] = Blue background (#356793), white text
          Full access - can manage team, coaches, all features

[Edit]  = Green background (#22c55e), white text
          Can create/edit practices, periods, templates

[View]  = Gray background (#9ca3af), white text
          Read-only access to team content

Badge styling:
- Border radius: 4px
- Padding: 2px 8px
- Font size: 11px
- Font weight: 600


STATUS BADGES
================================================================================

[Active]   = Green outline, green text
             Coach has accepted and is active

[Invited]  = Yellow/orange outline, yellow text
             Invitation sent, awaiting acceptance

[Inactive] = Gray outline, gray text
             Account inactive/disabled

Badge styling:
- Border: 1px solid
- Border radius: 4px
- Padding: 2px 8px
- Font size: 11px


SEARCH FUNCTIONALITY
================================================================================

+--------------------------------------+
| [O] Search by email or permission... |
+--------------------------------------+

Searches:
- Email address
- Permission level (admin, edit, view)
- Status (active, invited, inactive)

================================================================================

INTERACTIONS
================================================================================

1. Tap coach card -> Opens coach modal for editing
2. Tap [X] delete -> Shows remove confirmation
3. Tap FAB (+) -> Opens coach modal for inviting
4. Type in search -> Filters list
5. Tap [<] back -> Returns to Menu


PERMISSIONS
================================================================================

Admin users only can access this screen:
- Show FAB button
- Show delete buttons
- Can invite/edit/remove coaches

Non-admin users:
- Screen not accessible (hidden in menu)
- Or shows "Admin Only" message
