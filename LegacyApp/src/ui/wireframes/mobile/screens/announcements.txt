ANNOUNCEMENT CARD DETAIL
================================================================================

| message that gets truncated...       |
|                                      |
| Author Name • 2h ago          [HIGH] |  Priority badge
+--------------------------------------+

Read Announcement:
+--------------------------------------+
|    Announcement Title (Normal)       |  No dot, normal weight
|                                      |

| message that gets truncated...       |

+--------------------------------------+


UNREAD INDICATOR
================================================================================

[*] = Blue dot

Position: Left of title
Size: 8px diameter
Color: #356793 (primary blue)


PRIORITY BADGES
================================================================================

[HIGH] = Red background (#ef4444), white text
[MED]  = Pink background (#ef7b8f), white text
[LOW]  = Gray background (#9ca3af), white text

Badge styling:
- Border radius: 4px
- Padding: 2px 8px
- Font size: 10px
- Font weight: 600
- Uppercase text


TIMESTAMP FORMAT
================================================================================

Just now       -> "just now"
< 1 hour       -> "Xm ago"
< 24 hours     -> "Xh ago"
< 7 days       -> "Xd ago"
< 4 weeks      -> "Xw ago"
> 4 weeks      -> "MMM D"


HEADER WITH NEW BUTTON
================================================================================

+------------------------------------------+
|  [<] Menu        Announcements    [New]  |
+------------------------------------------+

[New] button:
- Only visible for admin users
- Text: "New" or "+" icon
- On tap: Opens announcement-create modal


INTERACTIONS
================================================================================

1. Tap announcement card -> Opens announcement-detail modal
   - Marks announcement as read (if unread)
2. Tap [New] button -> Opens announcement-create modal (admin)
3. Tap [<] back -> Returns to Menu tab


PERMISSIONS
================================================================================

Admin users:
- See [New] button in header
- Can create announcements
- Can edit/delete announcements

Non-admin users:
- No [New] button
- View-only access
- Can mark as read


CARD STATES
================================================================================

Unread:
- Title: Bold font weight
- Blue dot indicator
- Slightly elevated appearance

Read:
- Title: Normal font weight
- No dot indicator
- Standard card appearance

Touch state:
- Background darkens slightly
- Feedback on tap
