TEMPLATE CARD DETAIL
================================================================================

+--------------------------------------+
| Template Name                  [N]   |  [N] = FileText icon
| XX min  •  X activities              |  Duration + count
| [Tag1] [Tag2] [Tag3]                 |  Tags
|                               [X]    |  [X] = Trash delete
+--------------------------------------+

Card styling:
- Background: white
- Border radius: 8px
- Shadow: subtle drop shadow
- Padding: 16px
- Margin bottom: 12px

Icons:
- [N] FileText icon: shown if template has activities
- [X] Trash icon: delete button (red on tap)


TEMPLATE INFO LINE
================================================================================

"90 min  •  6 activities"

Format: {total_duration} min  •  {activity_count} activities

- Duration: Sum of all activity durations
- Count: Number of activities in template
- Separator: bullet point (•)


TAG DISPLAY
================================================================================

Tags shown inline (max 3 visible):
[Tag1] [Tag2] [Tag3]

If more than 3:
[Tag1] [Tag2] [+2]

Tag styling:
- Background: light themed color
- Text: darker matching color
- Border radius: pill shape
- Padding: 4px 8px

================================================================================

+------------------------------------------+
|            MODAL HEADER                  |
|  [X]        Create Template              |
+------------------------------------------+
|                                          |
|  Template Name                           |

|                                          |

|  +------------------------------------+  |

|  +------------------------------------+  |
|                                          |
+------------------------------------------+
|  TPFooterButtons                         |
|  mode="edit" | saveLabel="Create"        |

+------------------------------------------+

Props:
- mode="edit"
- onCancel={handleClose}
- onSave={handleCreate}
- saveLabel="Create"

- loading={isCreating}


EDIT TEMPLATE MODAL (Tap Card)
================================================================================

Tap template card -> Opens template modal for editing:

Route: /(modals)/template/[id].tsx

+------------------------------------------+
|            MODAL HEADER                  |

+------------------------------------------+

+------------------------------------------+
|  TPFooterButtons                         |
|  mode="edit" | saveLabel="Save"          |
|  [Cancel]                     [Save]     |
+------------------------------------------+

Props:
- mode="edit"
- onCancel={handleClose}

- saveLabel="Save"
- saveDisabled={!hasChanges}

INTERACTIONS
================================================================================

1. Tap template card -> Opens template/[id] modal for viewing/editing
2. Tap [X] delete -> Shows delete confirmation dialog (TPAlert)
3. Tap FAB (+) -> Opens template/new modal for creating
4. Type in search -> Filters list in real-time
5. Tap [<] back -> Returns to Menu tab


PERMISSIONS
================================================================================

canEdit = true:
- Show FAB button
- Show delete buttons
- Allow creating/editing templates

canEdit = false:
- Hide FAB button
- Hide delete buttons
- View-only mode
