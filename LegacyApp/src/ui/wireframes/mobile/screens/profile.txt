EDIT MODE FOOTER (TPFooterButtons)
================================================================================

Uses: TPFooterButtons component from @/components/tp

+------------------------------------------+
|  TPFooterButtons                         |
|  mode="edit" | saveLabel="Save"          |
|  [Cancel]                     [Save]     |
+------------------------------------------+

Props:
- mode="edit"
- onCancel={handleCancelEdit}
- onSave={handleSave}
- saveLabel="Save"
- saveDisabled={!hasChanges}
- loading={isSaving}


INTERACTIONS
================================================================================

1. Tap [Edit Profile] -> Enters edit mode (opens as modal or inline edit)
2. Tap [Save] in TPFooterButtons -> Saves changes, exits edit mode
3. Tap [Cancel] in TPFooterButtons -> Discards changes, exits edit mode
4. Tap photo option -> Camera/gallery/remove action
5. Toggle notification -> Updates preference
6. Tap [Logout] -> Shows confirmation, logs out
7. Tap User ID/Team ID -> Copies to clipboard
8. Tap [<] back -> Returns to Menu


VALIDATION (Edit Mode)
================================================================================

First Name:
- Required
- Min 1 character

Last Name:
- Required
- Min 1 character

Shows error messages below inputs if invalid
