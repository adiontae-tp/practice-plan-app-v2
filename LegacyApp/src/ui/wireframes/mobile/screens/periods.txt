TAG DISPLAY
================================================================================

Tag styling:

INTERACTIONS
================================================================================

1. Tap period card -> Opens period modal for viewing/editing
2. Tap [X] delete -> Shows delete confirmation (TPAlert)
3. Tap FAB (+) -> Opens period modal for creating new
4. Type in search -> Filters list in real-time
5. Tap [<] back -> Returns to Menu tab


PERMISSIONS
================================================================================

canEdit = true:
- Show FAB button
- Show delete buttons on user periods
- Allow editing periods

canEdit = false:
- Hide FAB button
- Hide delete buttons
- View-only mode
