# App Features

A comprehensive list of features available to users of the Practice Plan App.

## Core Features

### Practice Plan Management
- **Create Practice Plans** - Build detailed practice sessions with start time, duration, and multiple periods
- **Edit Plans** - Modify existing plans, reorder periods, adjust timing
- **Duplicate Plans** - Copy existing plans to create similar sessions quickly
- **Delete Plans** - Remove plans no longer needed
- **Plan Notes** - Add notes to entire practice plans
- **Plan Tags** - Categorize plans with tags for filtering

### Periods (Activities)
- **Add Periods** - Include timed periods/activities within a plan
- **Period Duration** - Set specific duration for each period
- **Period Notes** - Add detailed notes/instructions for each period
- **Period Tags** - Tag individual periods for categorization
- **Reorder Periods** - Drag and drop to reorder periods within a plan
- **Quick Add from Library** - Add pre-saved periods from period library

### Period Library
- **Create Periods** - Save reusable periods to library
- **Edit Periods** - Modify saved periods
- **Delete Periods** - Remove periods from library
- **Search Periods** - Find periods by name
- **Period Templates** - Pre-built periods with duration and notes

### Practice Templates
- **Create Templates** - Save complete practice structures as templates
- **Apply Templates** - Start new plans from existing templates
- **Edit Templates** - Modify saved templates
- **Delete Templates** - Remove templates no longer needed
- **Template with Periods** - Templates include pre-configured periods

## Calendar & Scheduling

### Calendar Views
- **Week View** - See practices for current week
- **Month View** - See practices across the month
- **Day Selection** - Click any day to see/add practices
- **Navigate Dates** - Move between weeks/months

### Schedule Management
- **View Upcoming** - See upcoming scheduled practices
- **Quick Create** - Create plans directly from calendar
- **Date/Time Selection** - Set precise start times for practices

## Team Management

### Team Settings
- **Team Name** - Set/update team name
- **Team Sport** - Select sport type
- **Team Logo** - Upload team logo
- **Team Colors** - Set primary/secondary brand colors
- **Custom Font** - Upload custom team font

### Coach Collaboration
- **Invite Coaches** - Send email invitations to coaching staff
- **Permission Levels** - Assign Admin, Edit, or View permissions
- **Remove Coaches** - Remove coaches from team
- **Resend Invites** - Resend pending invitations
- **Coach Status** - Track Active, Invited, Inactive status

## Organization

### Tags
- **Create Tags** - Create custom tags for categorization
- **Edit Tags** - Rename existing tags
- **Delete Tags** - Remove unused tags
- **Apply Tags** - Tag plans, periods, templates, files
- **Filter by Tag** - Find items by tag

### Files & Documents
- **Upload Files** - Upload PDFs, images, videos, documents
- **File Categories** - Organize by Playbook, Roster, Schedule, Media, Other
- **Folders** - Create folder structure for organization
- **File Search** - Find files by name
- **File Preview** - Preview files in-app
- **Download Files** - Download files to device
- **Favorites** - Mark files as favorites
- **Version History** - Track file versions
- **File Sharing** - Share files via link or with team members

## Communication

### Announcements
- **Create Announcements** - Post team announcements
- **Priority Levels** - Set Low, Medium, High priority
- **Rich Text** - Format announcements with rich text
- **Pin Announcements** - Pin important announcements
- **Read Tracking** - See who has read announcements
- **Push Notifications** - Send push notifications with announcements
- **Email Notifications** - Send email notifications
- **Unread Indicators** - See unread announcement count

## Analytics & Reports

### Practice Reports
- **Time Tracking** - Track total practice time
- **Period Analysis** - See time breakdown by period type
- **Tag Usage** - Analyze which tags are used most
- **Date Filtering** - Filter reports by date range
- **Practice Count** - Track number of practices

## PDF Export

### Plan Export
- **PDF Generation** - Export practice plans as PDFs
- **Multiple Templates** - Choose from PDF template styles
- **Share PDFs** - Share generated PDFs
- **Print Support** - Print-friendly PDF format

## Profile & Account

### User Profile
- **Profile Photo** - Upload profile picture
- **Personal Info** - Update name, email
- **Notification Settings** - Toggle push/email notifications
- **Account Info** - View account details (UID, team ID)

### Authentication
- **Email Login** - Sign in with email/password
- **Google Sign-In** - Sign in with Google account
- **Apple Sign-In** - Sign in with Apple ID
- **Password Reset** - Reset forgotten password
- **Account Creation** - Create new account

### Web App
