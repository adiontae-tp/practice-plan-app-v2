## Conversion Tracking (NON-NEGOTIABLE)

### You MUST track these events:

1. **Primary Conversion:** Pro/Team subscription purchase
   ```javascript
   // When user completes payment
   gtag('event', 'conversion', {
     'send_to': 'AW-CONVERSION_ID/CONVERSION_LABEL',
     'value': 29.99, // or 49.99 for Team
     'currency': 'USD',
     'transaction_id': subscriptionId
   });
   ```

2. **Secondary Conversion:** Clicked "Upgrade" button
   ```javascript
   // When user clicks any upgrade CTA
   gtag('event', 'conversion', {
     'send_to': 'AW-CONVERSION_ID/UPGRADE_CLICK_LABEL'
   });
   ```

3. **Micro-conversion:** Viewed pricing page
   ```javascript
   // When user visits /pricing or /upgrade
   gtag('event', 'view_pricing', {
     'send_to': 'AW-CONVERSION_ID/VIEW_PRICING_LABEL'
   });
   ```

### UTM Parameters
Add to all ad URLs:
```
?utm_source=google&utm_medium=cpc&utm_campaign=practice-planning&utm_content={ad_id}
```

This lets you track which ads drive conversions in Google Analytics.

## Weekly Optimization Routine

### Every Monday (15 minutes):

1. **Check conversions from last week**
   - Which keywords drove paid conversions?
   - Increase bids by 20% on converters
   - Pause keywords with 20+ clicks and 0 conversions

2. **Review Search Terms Report**
   - Add irrelevant searches as negative keywords
   - Add high-performing new terms as keywords

3. **Check Quality Score**
   - Improve ads/landing pages for keywords with QS < 7

### Every Friday (10 minutes):

1. **Budget pacing check**
   - Are you spending $3.33/day consistently?
   - If overspending, lower bids by $0.10
   - If underspending, raise bids by $0.10

2. **Ad performance**
   - Pause ads with CTR < 2%
   - Duplicate high-performing ads

## When to Pause Everything

**Stop and reassess if:**
- After 100 clicks with 0 conversions ‚Üí Landing page problem
- Cost per click exceeds $3 ‚Üí Wrong keywords, too competitive
- CTR below 1% after 30 days ‚Üí Ad copy not resonating
- Bounce rate > 80% on landing page ‚Üí Traffic quality issue

**Troubleshooting:**
- High CPC? ‚Üí Target longer-tail keywords
- Low CTR? ‚Üí Rewrite ad copy to match search intent
- Clicks but no conversions? ‚Üí Fix landing page (pricing too high? CTA unclear?)

## Questions to Answer First

Before you spend a dollar, clarify:

1. **What's your current free ‚Üí paid conversion rate organically?**
   - If 0%, fix your product first before advertising
   - If 1-5%, Google Ads can scale that
   - If >5%, you're ready to scale

2. **What's the lifetime value (LTV) of a Pro customer?**
   - Average months a customer stays √ó $29.99
   - If 6 months = $180 LTV
   - You can spend up to 1/3 of LTV to acquire ($60 max)

3. **Can you track which users came from Google Ads in your database?**
   - Need UTM tracking in Firebase/Stripe
   - Need to attribute conversions back to specific ads

4. **Do you have testimonials/social proof for landing page?**
   - Get 3-5 coach testimonials before launching
   - "As a high school basketball coach, this app saves me 5 hours a week"

5. **Is your pricing competitive?**
   - Research: TeamSnap, Hudl, CoachNow pricing
   - Make sure $29.99 is justified vs. competitors

6. **Do you have a free trial for Pro/Team?**
   - Huge conversion boost if you offer "Try Pro free for 14 days"
   - Makes the upgrade decision much easier
